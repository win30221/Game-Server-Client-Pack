Thank you for using Localization!

Those scripts are base on uGUI.
The asset will provide you simple switch language, including Text, ...etc.

----------------------------
 How To Use Localization
----------------------------

1. Do it before using
   (1) Prepare a ttf font.
   (2) Fill up the "Language" table, column 0 is unique key to identify.
2. Create a gameobject and attached "LocalizationManager" script.
3. Attached "LocalizationText" script to the Text of uGUI and fill up the key into the name that you want to show.
4. If you want to switch the language, you can use SwitchLanguage() of the "LocalizationManager".
